# Isometric World Engine

Процедурний ізометричний рушій, написаний на C# (WinForms/GDI+) з підтримкою планетарного масштабу (світ розміром 1,000,000 x 1,000,000 тайлів).

## 📂 Структура проєкту

### `IsometricWorld.Core`
Базова логіка рушія, незалежна від конкретного фреймворку відображення.
- **`Generation/`**
  - `WorldGenerator.cs`: Алгоритм процедурної генерації на основі 3-октавного Fractal Brownian Motion.
  - `PerlinNoise.cs`: Реалізація шуму Перліна для генерації рельєфу та вологості.
- **`Rendering/`**
  - `IsometricRenderer.cs`: Основний рендерер GDI+. Реалізує Painter's Algorithm для Z-сортування тайлів, пропсів та будівель.
  - `IsometricCamera.cs`: Управління камерою, плавний рух та зворотна математика (`ScreenToWorld`) для кліків миші.
- **`UI/`**
  - `UIManager.cs`: Кастомна UI-система на базі GDI+, яка працює поверх ізометричного світу (без стандартних контролів для уникнення мерехтіння). 
- **`World/`**
  - `ChunkManager.cs`: Управління чанками, стримінг світу, завантаження та вивантаження з пам'яті.
  - `Chunk.cs` / `Tile.cs`: Дані світу `-та окремих клітинок.
- `WorldEngine.cs`: Головний клас-координатор, що об'єднує ChunkManager, Renderer, Camera, логіку вільного будівництва та генерацію Глобальної Карти (Minimap).

### `IsometricWorld.Demo`
Точка входу та віконний застосунок.
- `GameWindow.cs`: Основний цикл гри (Game Loop), обробка інпуту (клавіатура, миша), виклик оновлень (`Update`) та перемальовки (`OnPaint`).
- `AssetManager.cs`: Менеджер ресурсів (раніше використовувався для текстур, зараз рушій переведено на GDI+ примітиви для швидкодії).
- `Program.cs`: Entry point.

### `IsometricWorld.Tests`
Юніт-тести для перевірки цілісності логіки рушія (наприклад, перевірка ChunkManager).

## 🚀 Поточний етап розробки

**Етап:** _"Фундамент вільного будівництва та кастомний UI"_

### Що вже реалізовано (Features):
1. **Екстремальна Процедурна Генерація**:
   - Масштаб світу: 1,000,000 на 1,000,000 тайлів.
   - Складна генерація континенту з використанням багатошарового FBM шуму та квадратичного Falloff.
   - 6 збалансованих біомів (Океан, Пляж, Рівнини, Ліс, Гори, Річки).
   - Рандомізація світів "на льоту" (натискання клавіші `R`).
2. **Оптимізований Рендеринг**:
   - Повністю геометричний рендер (без зовнішніх текстур).
   - Painter's Algorithm із Z-сортуванням.
   - Глобальна карта (на клавішу `M`) з динамічним кешуванням у `1000x1000` пікселів, що дозволяє миттєво переглядати світ планетарного масштабу без просадок FPS.
3. **Взаємодія та Камера**:
   - Плавний рух камери.
   - Зворотна ізометрична проекція (клік по екрану → світові float-координати).
4. **Custom UI System**:
   - GDI+ інтерфейс поверх гри (панель інструментів, спливаюче меню з тінями та градієнтами).
   - Математичний Bounding Box для перехоплення кліків.
5. **Вільне Будівництво (Free-form Building)**:
   - Розміщення 3D-блоків за межами тайлової сітки (вільні координати).
   - **Оптимізовані колізії**: циліндрична (радіусна) перевірка відстані між об'єктами, що ітерується лише по завантажених чанках (O(N) від активних об'єктів).
   - **Управління пам'яттю**: будівлі зберігаються у структурі відповідних чанків і вивантажуються/завантажуються з пам'яті разом із ними.
   - Точне Z-сортування будівель відносно ізометричної перспективи екрану.

## 🕹️ Управління
- **W, A, S, D** або **Стрілки**: Рух камери.
- **M**: Відкрити/Закрити глобальну карту світу (червона точка — позиція камери).
- **R**: Згенерувати новий випадковий світ.
- **ЛКМ по Кнопці Будівництва (зліва внизу)**: Відкрити меню блоків.
- **ЛКМ по "Block" у меню**: Активувати режим будівництва (молоток стане зеленим).
- **ЛКМ по світу (в режимі будівництва)**: Поставити блок.
